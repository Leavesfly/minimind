# MiniMind3 技术 Wiki

> **如果 GPT-4 是一栋摩天大楼，这份 Wiki 就是它的建筑图纸——让你从地基到天线，一层一层看懂大楼是怎么盖起来的。**

MiniMind3 是一个完全从 0 实现的极小语言模型项目（Dense 64M / MoE 198M-A64M），完整覆盖 LLM 训练全流程。本 Wiki 面向**想真正读懂每一行代码**的同学，用第一性原理和生活类比，从"为什么"出发解释每个设计决策。
